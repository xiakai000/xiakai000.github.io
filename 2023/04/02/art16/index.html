计算机网络做题解析六

发表于2023-04-02|更新于2023-07-03
|字数总计:500|阅读时长:1分钟|阅读量:

某主机的IP地址202.117.131.12/20,其子网掩码是（）。

255.255.248.0
255.255.240.0
255.255.252.0
255.255.255.4
解析：20即说明前20位为网络号，那么后12位为主机号，所以前20位为1，后12位为0的IP地址为255.255.240.0

网段地址 154.27.0.0 的网络。若不做子网划分，能支持（）台主机

254
1024
65,534
16,777,206
解析：

B类地址有16位是主机号的位数
前16位为网络号，后16位为主机号，那么2^16 = 65536,除去全0和全1的即65534

办公室的一台计算机只能使用IP地址而无法使用域名访问外部网站，造成这种情况的原因不可能是（ ）

该计算机设置的DNS服务器没有提供正常域名解析服务
该计算机与DNS服务器不在同一子网
提供被访问服务器名字解析的DNS服务器有故障
该计算机发起的目的端口为53的访问请求被限制
解析：可能是DNS服务器配置错误或者故障了，所以可能是计算机设置的DNS服务器没有提供正常域名解析服务，也可能是提供被访问服务器名字解析的DNS服务器有故障，而53号服务器是DNS服务器的访问请求，被限制也可能使无法访问，计算机和DNS服务器不在一个子网不受影响

一台刚刚接入互联网的WEB服务器第一次被访问到时，不同协议的发生顺序是下面中的____。

ARP -> DNS -> HTTP
ARP -> HTTP -> DNS
DNS -> HTTP -> ARP
DNS -> ARP -> HTTP
HTTP -> ARP -> DNS
HTTP -> DNS -> ARP
解析：第一次接入网络时要先将mac地址通过ARP 发送到网关，在网关记录ip 和 mac 的对应信息。访问服务器，访问的是域名，所以先通过DNS查找域名对应IP，再通过HTTP访问IP对应服务器
